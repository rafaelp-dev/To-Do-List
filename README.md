# To-Do List

Projeto simples de lista de tarefas desenvolvido em Java, utilizando linha de comando.

## Funcionalidades

- Adicionar novas tarefas
- Listar tarefas existentes
- Editar tarefas
- Remover tarefas
- Marcar tarefas como concluídas

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git
